# Portfolio Project Evidence Reviews

This folder contains one project evidence review file per portfolio project.

Each file separates:

- Implemented features
- Planned / not proven items
- Safe claims
- Claims to avoid
- CV bullets
- LinkedIn wording
- Interview talking points
- Next sprint recommendation

These files are **evidence reviews**, not marketing exaggeration.

GitHub/ZIP review is not the same as local validation. Local validation remains **pending** unless terminal output proves it.

SweetCart Bakery is not listed separately because it is part of BakeOps context, not a separately reviewed project here.
