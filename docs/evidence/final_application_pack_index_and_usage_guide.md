## Recommended Application Workflow

1. Check role fit and skip obvious mismatches (location, sponsorship, unsupported must-haves).
2. Open the locked CV: **Aminul_Islam_Data_Analyst_CV**.
3. Use Step 3 to paste and analyse the JD (intake form, requirement extraction, apply/skip rule).
4. Use Step 2 to choose one role-specific cover letter variant.
5. Use Step 1 if a general master cover letter is enough (simple or fallback applications).
6. Use Step 4 for recruiter messages, LinkedIn DMs, follow-ups, and replies.
7. Run claim-safety checks before sending (see Claim-Safety Master Checklist).
8. Save application notes in CareerFunnel Tracker manually.

## Portfolio Evidence Priority

### 1. BakeOps Intelligence

- **Best use:** Data Analyst, BI/Reporting Analyst, Operations Analyst roles; operational KPIs and profitability insight.
- **Safe wording:** Portfolio project with KPI surfaces, BI-ready CSV exports, metric documentation, and rank-inversion insight (revenue vs margin rank).
- **Avoid:** Production users, live SaaS, external client delivery claims.

### 2. CareerFunnel Tracker

- **Best use:** Data Analyst, Operations Analyst, Analytics Engineer stretch; structured metrics, data quality, workflow governance.
- **Safe wording:** Django portfolio with funnel metrics, Evaluation Queue, data-quality warnings, manual assisted-intake evidence, and 249 tests.
- **Avoid:** Auto-apply, scraping, external AI, live job-platform integration claims.

### 3. TradeIntel 360

- **Best use:** BI/Reporting Analyst, FinTech / FX / Payments Analytics; post-trade KPI and export narratives.
- **Safe wording:** Portfolio project calculating trading KPIs from CSV uploads with documented methodology.
- **Avoid:** Institutional trading system, live brokerage, or production trading infra claims.

### 4. DataBridge Market API

- **Best use:** FinTech analytics, Analytics Engineer stretch, Junior Data Engineer stretch (junior, analyst-friendly JDs).
- **Safe wording:** Portfolio market-data ingestion, run logging, normalised storage, and inspection workflow.
- **Avoid:** Production pipelines, cloud platforms, enterprise API service claims.

### 5. MarketVista Dashboard

- **Best use:** FinTech monitoring secondary mention only when JD asks for watchlists/alerts/monitoring.
- **Safe wording:** Portfolio monitoring dashboard logic with sparse-intraday handling.
- **Avoid:** Live trading decisions, production alerting, investment advice claims.
