## Daily Live Application Checklist

Use every application day. Manual only.

- [ ] Search suitable roles (target roles first; stretch only if worth time).
- [ ] Run role-fit gate (Sprint 27A): title, location, seniority, tools, blockers, sponsorship, scam risk.
- [ ] Save job URL and full JD.
- [ ] Assign fit rating and priority (Sprint 27B Fit Rating / Priority Rules).
- [ ] Select Application Pack document (Step 1-5 per situation).
- [ ] Attach or reference `Aminul_Islam_Data_Analyst_CV` (no CV rewrite).
- [ ] Draft cover letter (Step 3 if full JD; Step 2 if role clear; Step 1 fallback).
- [ ] Run Claim-Safety Closure Checklist (below) before send.
- [ ] Submit manually (portal or email); no auto-apply.
- [ ] Record application in CareerFunnel Tracker (Sprint 27B field mapping).
- [ ] Record cover letter source, project evidence (max two), claim-safety result.
- [ ] Set follow-up date if Submitted or Outreach Sent (Step 4: 3-day or 7-day).
