How to assess this portfolio project

CareerFunnel Tracker is a manual, advisory, deterministic job-search analytics app for one user. Review saved records, rule-based guidance, and evidence docs - not live SaaS automation.

  1. Start here on the dashboard for today signals, pipeline health, and evidence readiness gaps.
  2. Review applications, follow-ups, interviews, daily logs, and weekly reviews you control manually.
  3. Open Funnel Metrics for reporting and export evidence.
  4. Open Skill Intelligence Dashboard for saved skill gaps and advisory planning sections.
  5. Read docs/evidence/evidence_index.md and sprint evidence under docs/evidence/ for the implementation trail.

Deliberately not implemented: automatic application submission, automatic outbound email, Gmail/Calendar/OAuth integration, scraping, automatic CV rewriting, automatic interview prep generation, AI prediction claims, background polling, live SaaS users, production deployment, or billing.
