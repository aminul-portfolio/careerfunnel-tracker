Job AI Capability Match

Rule-based keyword matching report for manual review.

This page shows how demo job description text maps to Sprint 53 AI capability categories through deterministic keyword detection. Phase 2 uses sample text only.

This is a rule-based keyword matching report for manual review. It is not predictive hiring AI. It does not use external AI APIs. It does not scrape jobs. It does not automate applications.

Skill gap signals are advisory only. They indicate learning priorities, not current proficiency. Before adding a skill to your CV or public profile, ensure it is supported by project evidence, tests, screenshots, or prior work experience.
