Skill gaps workflow (manual and advisory)

Advisory only. Based on saved skill-gap records. Review and decide manually - each section below is read-only and uses existing ApplicationSkillGap fields only.

  1. Manual action plan - suggested next steps from unresolved gaps.
  2. Manual learning plan - learning focus and suggested practice prompts.
  3. Manual evidence readiness - which gaps need stronger proof ideas.
